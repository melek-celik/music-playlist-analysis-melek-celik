# music-playlist-analysis-melek-celik
Proje Amacı: Bu proje, Python kullanarak bir müzik çalma listesini analiz etmek amacıyla oluşturulmuştur.  

Metot Açıklamaları:

get_total_duration: Listenin toplam süresini hesaplar.  

get_most_played_song: En popüler şarkıyı bulur.  

get_average_duration: Şarkıların ortalama süresini verir.  

print_playlist: Listeyi tablo formatında yazdırır.  

Çalıştırma Bilgisi: python main.py komutu ile terminal üzerinden çalıştırılabilir.
